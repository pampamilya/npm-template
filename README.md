## npm-template

> This is a NPM template for creating packages.

### Installation 🦸‍♂️

```sh
# Install with NPM
npm i --save @pampamilya/npm-template

# Or with Yarn
yarn add @pampamilya/npm-template
```

### Usage 🐱‍👓

```js
import { pampamilya } from "@pampamilya/npm-template";

console.log(pampamilya());
```
